# Store Sales Forecasting

Simple time series forecasting project for the Kaggle Store Sales competition.

The goal is to explore the store sales data, build a forecasting model, and create a submission file for Kaggle.

## Links

- Kaggle notebook: https://www.kaggle.com/code/mdshafayeturrahman/store-sales-forecasting-eda-to-submission
- Kaggle dataset: https://www.kaggle.com/competitions/store-sales-time-series-forecasting/data

## Project Structure

```text
.
├── data/
│   ├── raw/          # Original Kaggle files
│   └── processed/    # Cleaned or prepared data
├── models/           # Saved model files
├── notebooks/        # EDA and experiments
├── reports/
│   └── figures/      # Charts and images
├── src/              # Reusable Python code
├── README.md
├── requirements.txt
└── .gitignore
```

## How to Use

1. Download the competition data from Kaggle.
2. Put the original files in `data/raw/`.
3. Use notebooks for EDA and experiments.
4. Put reusable code in `src/`.
5. Save final predictions as `submission.csv`.
